Welcome to Event Ticketing System

API Endpoints

Method Endpoint Description
GET /api/events Return all events
GET /api/events/:id Return details of a single event
GET /api/events?category= Filter events by category (optional)
GET /api/events?date= Filter events by date (optional, e.g., YYYY-MM-DD)
POST /api/auth/register Register a new user
POST /api/auth/login Authenticate a user and return JWT token
POST /api/events Create a new event (admin only)
POST /api/bookings Book tickets for an event (user only)
PUT /api/events/:id Update an event (admin only)
DELETE /api/events/:id Delete an event (admin only)
